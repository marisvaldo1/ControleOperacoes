CryptoModalHeader — Demonstração

↑ Cabeçalho padrão gerado por CryptoModalHeader.mount()

Os filtros acima são funcionais. Selecione um período, status ou tipo e observe o callback.

Código de integração

// 1. No HTML do seu modal:
<div id="meuModalHeader"></div>

// 2. No JS da sua janela:
const header = CryptoModalHeader.mount('#meuModalHeader', {
  title:         'Dashboard Analítico · Crypto',
  icon:          '⚡',
  defaultPeriod: 'today',
  closeModalId:  'meuModalId',
  actionLabel:   'Análise Detalhada',   // opcional
  actionId:      'btnAnaliseDetalhada', // opcional
  onFilter: (state) => {
    // state = { period, status, tipo, asset, corretora }
    render(state);
  },
  onRefresh: async () => {
    const data = await fetch('/api/crypto').then(r => r.json());
    window.cryptoOperacoes = data;
    render(header.getState());
  },
  onAction: () => {
    abrirAnaliseDetalhada();
  },
  showTotals: true,
});

// 3. Após carregar/filtrar operações:
header.setOps(todasOps, opsFiltradas);

// 4. Para atualizar o relógio (chamado no final de cada render):
header.tick();
